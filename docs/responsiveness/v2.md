# Mobile UI Optimization Notes

## /chat/chatid

* **Message Input Responsiveness**

  * Force long messages without spaces to wrap or truncate at the max width to prevent overflow on mobile.

* **MessageReactionPicker and Notification Popover**

  * Ensure these components are responsive.
  * Prevent overflow outside the screen on small devices.
  * On mobile, position `MessageReactionPicker` at the center of the screen, above the text input for better accessibility and usability.
  * Make the notification popover responsive as well; it currently overflows the screen when opened on small devices.
