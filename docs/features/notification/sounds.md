### Notification Sound
- Add a sound when receiving:
  - New messages
  - Other notifications (if applicable)
  - Users can select a notification sound. Check the frontend/public/sounds/notification/... to check all the 6 notification sounds option.
  - Notification sound is on by default. But can be off under settings>notifications>below push notificatinos.
  - Notification sound should still be able to play even if the user is outside the tab, as long as he receives a notification.
